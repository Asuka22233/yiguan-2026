# 仪光赤心 · 2026寒假

仪光赤心实践队 2026 寒假社会实践照片墙网站。

## 功能

- **照片墙**：多行随机滚动展示，鼠标悬停暂停
- **全部照片**：按分类（实践活动 / 红色教育 / 团队风采 / 社区服务）筛选浏览
- **照片详情**：点击照片查看大图、描述、时间、地点，支持键盘左右切换
- **实践研究成果**：链接至 AI 红色时光对话机、红色文化地图两个项目
- **响应式**：适配桌面端与移动端

## 文件结构

```
index.html   主页面（HTML + CSS + JS）
photos.js    照片数据（独立维护）
README.md    说明文件
```

## 使用

直接用浏览器打开 `index.html` 即可预览。部署到 GitHub Pages 或任意静态托管服务即可上线。

### 替换照片

编辑 `photos.js`，修改每张照片的 `src`、`title`、`desc`、`tag`、`date`、`location` 字段即可。

## 技术栈

- 纯 HTML / CSS / JavaScript，无外部依赖
- CSS 自定义属性（变量）控制主题配色
- CSS 动画实现滚动照片墙与粒子效果
